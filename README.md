# セラピスト向け顧客管理アプリ (Therapist CRM)

セラピストがスマートフォン上で顧客情報、来店履歴、施術内容、お支払金額をスムーズに確認・記録できる、スマホ最適化の顧客管理システムです。

## 1. 特徴・機能
- **スマホ特化UI**: 4つの情報を瞬時に切り替えるタブコントロールを配置。
- **LocalStorage 連携**: データベース不要で、ブラウザ内で完結する安全なデータ永続化。
- **検索フィルタ**: 顧客名やよみがなによるインクリメンタルなあいまい検索。
- **自動ログ分類・保存**: 開発時のAI対話ログをアクセス元URLごとに自動で階層分類して保存するデベロッパーブリッジを搭載。

## 2. クイックスタート
### 依存関係の起動
```bash
npm run dev
```
これにより、以下のサーバーが並行起動します：
- **フロントエンドポータル / アプリケーション (3005番ポート)**: `browser-sync` によるホットリロードサーバー
- **ログ保存・プロキシブリッジ (3002番ポート)**: チャット履歴を分類保存する `bridge.js`
