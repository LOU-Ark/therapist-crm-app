# ISSUE-002: アプリケーションヘッダーへのテーマ調整スライダー追加

## 1. インシデント概要
* **発生日**: 2026-07-20
* **分類**: MINOR (v1.2.0 - 新機能・UI拡張)
* **症状**: アプリケーション本体（クライアントモード）側でも、ポータルやゲートウェイと同じくテーマ（Dark/Light）のブレンドスライダーによる動的な輝度変更を行いたいという要望。
* **影響度**: 低（ユーザビリティおよびデザイン性向上のためのUI拡張）

## 2. 根本原因分析 (RCA)
- 要件策定時、ポータルとゲートウェイにはテーマスライダーを配置していたものの、アプリ本体側は静的インジケーター表示に留めており、直感的なリアルタイム切替機能が欠落していました。

## 3. 是正処置（修正内容）
- アプリ本体のヘッダー [index.html](file:///c:/Users/Tan0Ry0/Desktop/helthcareapp/product/index.html) のインジケーターを、インライン型のテーマ調整スライダー（`theme-slider`）へ変更。
- スクリプト [ui.js](file:///c:/Users/Tan0Ry0/Desktop/helthcareapp/product/js/app/ui.js) にて LocalStorage 内の `theme-blend` 値を共有ロード・同期し、スライド時に CSS 変数 `--theme-blend` を動的に更新する制御を追加しました。
