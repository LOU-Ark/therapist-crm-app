# ISSUE-006: 顧客一覧の四角いカードグリッド化およびカルテ詳細の全画面表示化

## 1. インシデント概要
* **発生日**: 2026-07-20
* **分類**: MINOR (v1.5.0 - UI/UX刷新・デザイン最適化)
* **症状**: 
  - 顧客一覧を従来の縦長リスト形式から、画面全体を贅沢に使った「四角いカード」が並ぶグリッドレイアウトに変更したい。
  - 各カードは、顧客のソウルカラーで視覚的に識別・分類できるように配色を適用したい。
  - カードクリック時は、その顧客のカルテ（詳細情報）を画面いっぱいに表示し、情報の記述・編集に集中できるモードへ遷移させたい。
* **影響度**: 低（CRMシステムとしての操作効率およびデザイン性の劇的向上）

## 2. 根本原因分析 (RCA)
- これまでは縦並びのシンプルな顧客リストを配置しており、大画面のスペースが活用できていませんでした。また、詳細表示時も一覧を横に残していたため、カルテの閲覧・記述に集中しづらいレイアウトとなっていました。

## 3. 是正処置（修正内容）
1. **顧客カードのグリッドレイアウト＆ソウルカラーバッジ化**: 
   - [index.css](file:///c:/Users/Tan0Ry0/Desktop/helthcareapp/product/css/index.css) にて `.customer-list` を CSS Grid に変更し、レスポンシブな複数列グリッド（`repeat(auto-fill, minmax(200px, 1fr))`）を定義。
   - 13色のソウルカラーに対応する `.soul-card-{color}` クラス群を定義。各ソウルカラーの HSL グラデーションを微量ブレンドし、ホバー時に発光する美しいガラスモフィズムカードへ刷新。
2. **全画面カルテ記述モード (`detail-mode`)**: 
   - [ui.js](file:///c:/Users/Tan0Ry0/Desktop/helthcareapp/product/js/app/ui.js) にて顧客カードクリック時、`.workspace-grid` に `.detail-mode` を追加。
   - [index.css](file:///c:/Users/Tan0Ry0/Desktop/helthcareapp/product/css/index.css) で `.detail-mode` 時に自動的に一覧側（左列）を非表示にしてカルテ詳細（右列）を全画面（100%幅）で表示する仕様に改修。カルテを閉じる（×）と元のグリッド一覧にスムーズに復帰する制御を実装しました。
